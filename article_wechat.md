---

## ✨ 核心功能

1. **书单自动同步**：将微信读书的所有存书一键搬运至飞书多维表格。
2. **支持断点续写**：智能识别已有记录，无需重复导入，节省时间。
3. **阅读进度更新**：实时刷新当前阅读百分比，进度一目了然。

## 🛠️ 图文版操作讲解

使用前请确保本地已安装 **Node.js** 环境。

### 第一步：复制飞书模板
[点击这里访问模板链接](https://c1w7wycgdlz.feishu.cn/base/SwaXbAx33af2iJs33zXcItyRnlE?from=from_copylink)

点击页面右上角的**“创建副本”**，将模板保存到你自己的飞书空间。

[图片]

### 第二步：源码下载并运行
1. **克隆项目**：
   ```bash
   git clone -b master https://github.com/Luchen-0420/vue-weread-feishu/tree/master
   ```
2. **安装依赖**（建议以管理员权限打开终端）：
   ```bash
   npm i -S @lark-base-open/node-sdk
   ```
   [图片]
3. **启动服务**：
   ```bash
   npm run dev
   ```
   看到如下图提示即表示服务启动成功：
   [图片]

### 第三步：在飞书表格中添加插件
在多维表格中点击**“自定义插件”**，填写你本地启动的服务地址（通常是 localhost 地址）。
*注：具体步骤若有疑问可参考 GitHub 详细说明。*

[图片]

### 第四步：获取微信读书 Cookie
1. 浏览器登录[微信读书网页版](https://weread.qq.com/)。
2. 按 **F12** 打开开发者工具，切换到 `Network` (网络) 标签。
3. 刷新页面，找到 `weread.qq.com` 的请求。
4. 在 `Headers` 中找到 `Cookie` 字段，完整复制其内容。

[图片]

### 第五步：首次同步书架
作为一个新用户，在插件界面直接点击**“同步书架”**即可完成初始化。

[图片]

### 第六步：更新书架（增量更新）
如果书架新增了书籍，请按以下步骤操作：
1. 点击**“同步表格”**：获取已有记录。
2. 点击**“同步书架”**：监测新增书籍。
3. 点击**“更新表格”**：将新增书籍写入飞书。

[图片]

### 第七步：更新阅读进度
你可以通过 `bookid` 或 **书名** 筛选指定书籍，点击**“更新进度”**，让飞书里的进度条动起来！

[图片]

### 第八步：制作精美画册
1. 点击“自定义插件” -> “获取授权码”。
2. 在插件市场搜索“链接转附件”，填写授权码并关联相应字段。
3. 运行完成后，点击**“画册视图”**即可看到视觉效果。
