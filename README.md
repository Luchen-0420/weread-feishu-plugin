> **既然选择了阅读，那就让数据说话。**  
> 这是一款专为阅读爱好者打造的飞书多维表格插件，旨在将微信读书的阅读轨迹、书柜数据及读书笔记无缝同步至飞书，构建属于你的私人数字图书馆。

---

## ✨ 核心功能

*   🔄 **书单自动同步**：一键将微信读书书架同步至飞书，支持书籍分类、作者、封面等元数据集成。
*   ✍️ **笔记一键导出**：**[NEW]** 支持将书籍划线、想法一键生成精美的 Markdown 笔记，并导出为文件。
*   🔖 **断点续写更新**：智能识别已同步书籍，仅更新变动内容，高效且节省流量。
*   📈 **阅读进度追踪**：可视化呈现书籍阅读百分比及阅读时长，让进步清晰可见。
*   🖼️ **画册视图集成**：结合飞书多维表格的画册视图，自动生成一眼惊艳的个人书柜展示。
*   🍪 **Cookie 智能管理**：深度支持 CookieCloud，告别手动抓包的繁琐（高级版功能）。
